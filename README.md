SkillProof – Verified Skills. Trusted Talent

From claims to proof — redefining how skills are verified and trusted in hiring.

Overview

**SkillProof** is a proof-based hiring platform that replaces traditional resumes with **verified skill evidence**.

Instead of claiming skills, users:

* Solve real-world tasks
* Get evaluated using AI + peer validation
* Build a trusted, proof-backed portfolio

🎯 Goal: Eliminate fake claims and bring **trust, transparency, and credibility** to hiring.

 Problem

>> Resumes are often exaggerated or misleading
>> Recruiters cannot verify real skills before hiring
>> No proof of ability until actual work begins

This creates a **trust gap in hiring**

---

Solution

SkillProof bridges this gap by:

Providing **real-world skill challenges**
 *Verifying work using **AI + peer review**
*Generating **proof-based badges & portfolios**
 Enabling recruiters to hire based on **evidence, not claims**

---

Key Features

AI Skill Verification

Automated evaluation of submissions with intelligent scoring.

 Proof-Based Badges

Earn verifiable credentials backed by real work.

 Auto Portfolio Builder

Dynamic portfolio showcasing verified skills.

 AI Interview Simulator

Practice interviews with instant feedback.

 Real-World Task Engine

Industry-level challenges simulating real jobs.

 Tamper-Proof Certificates

Blockchain-based verification to prevent fraud.

Trust Score System

Performance + peer validation combined into a credibility score.

 Skill Growth Timeline

Track progress and improvement over time.

 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/skillproof.git
cd skillproof
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file and add:

```env
REACT_APP_SUPABASE_URL=your_url
REACT_APP_SUPABASE_KEY=your_key
OPENAI_API_KEY=your_key
```

### 4. Run the app

```bash
npm start
```
